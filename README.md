# CONDITIONAL GAN - MODEL PIX2PIX

### Architecture components

- Generator => Encoder (downsampling) - Decoder (upsampling)

- Discriminator => Building blocks NN architecture

- Frontend => Python + Streamlit

- Backend => Python + Django


### Set of instructions

1) Create and activate python environment

$ python -m venv pix_env  
$ .\pix_env\Scripts\Activate.ps1
  
2) Open the Google Colab repository and execute the jupyter notebook

https://colab.research.google.com/drive/1JQCU-R-YGYO_skb9nTT5R-s1bhnpfIwG#scrollTo=ITZuApL56Mny   

3) Download this repository in local computer  
  
4) Install the necessary python libraries  
$ pip install -r requirements.txt  

5) Command to run the application

$ streamlit run app.py
